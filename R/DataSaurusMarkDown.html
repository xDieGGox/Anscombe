Resumen del dataset

Aquí presentamos un resumen del dataset DataSaurus, en sí lo que se hace es calcular las estadísticas para cada subgrupo del dataset. Con esto, podemos observar que las estadísticas son prácticamente iguales, aunque como vamos a ver posteriormente en esta práctica, en la visualización son completamente diferentes. De esta forma, demuestran lo que nos quiere decir el Cuarteto de Anscombe moderno. Para obtener estos resultados, se aplicaron las funciones group_by(dataset) para agrupar los datos según cada conjunto y summarize() para calcular la media, desviación estándar y correlación.

datasaurus_dozen %>%
  group_by(dataset) %>%
  summarize(
    mean_x = mean(x),
    mean_y = mean(y),
    std_dev_x = sd(x),
    std_dev_y = sd(y),
    corr_x_y = cor(x,y)
  )

Mostramos un resumen grafico de el Dataset

A continuación, mostramos 13 gráficos del dataset con la ayuda de la librería ggplot2, para poder visualizar que los gráficos de cada subgrupo del dataset son diferentes entre sí, a pesar de que sus estadísticas numéricas son iguales.

ggplot(datasaurus_dozen, aes(x=x, y=y, colour = dataset))+
  geom_point()+
  theme_void()+
  theme(legend.position = "none")+
  facet_wrap(~dataset, ncol = 3)

Resumen estadístico de CIRCLE

En esta parte, lo que realizamos es separar solo el dataset circle y obtener las estadísticas del mismo. Así, podemos identificar algunas métricas como la media, varianza y desviación estándar. En sí, es lo mismo en cuanto a datos estadísticos que los otros subgrupos.

datasaurus_dozen %>%
  filter(dataset == "circle") %>%
  summarize(
    mean_x = mean(x),
    mean_y = mean(y),
    std_dev_x = sd(x),
    std_dev_y = sd(y),
    corr_x_y = cor(x, y)
  )

Grafica de CIRCLE

En esta parte, lo que se realiza es mostrar los puntos del dataset circle, para poder visualizar estos datos. Se puede observar que tienen forma de elipse o circunferencia, lo cual no es evidente solo con los valores estadísticos. Usamos la librería ggplot2 para generar el gráfico, lo que nos ayuda a entender la distribución de los datos, que en este caso forma un círculo. De esta manera, podemos complementar el análisis numérico anterior con una visualización que revela la verdadera estructura de los datos, la cual no es evidente únicamente a través de estadísticas.

ggplot(filter(datasaurus_dozen, dataset == "circle"), aes(x = x, y = y)) +
  geom_point(color = "#2C3E50") +
  theme_minimal() +
  labs(title = "Dataset: Circle")

Regresión lineal sobre CIRCLE

En esta parte aplicamos una regresión lineal en el dataset circle. Lo que se realizó fue filtrar los datos y luego, con la función lm(), pudimos obtener la recta que mejor se ajusta a los puntos. Luego, con ggplot2, mostramos los puntos y también, en rojo, la línea que representa la regresión lineal. Esto nos sirvió para poder comparar el gráfico que realizamos a mano, solo viendo la forma del círculo sin usar valores numéricos, de tal forma que lo podemos comparar para ver si se parece o no al que hicimos de forma empírica.

# Filtramos el dataset
circle_data <- datasaurus_dozen %>%
  filter(dataset == "circle")

# Ajustamos el modelo de regresión lineal
modelo <- lm(y ~ x, data = circle_data)

ggplot(circle_data, aes(x = x, y = y)) +
  geom_point(color = "#2C3E50") +
  geom_smooth(method = "lm", se = FALSE, color = "red", linewidth = 1) +
  theme_minimal() +
  labs(title = "Regresión lineal - Dataset: Circle")
## `geom_smooth()` using formula = 'y ~ x'

Conclusión

Esta práctica nos permitió entender que, aunque varios conjuntos de datos pueden tener estadísticas similares, su forma visual puede ser completamente distinta. Por eso, siempre es importante complementar el análisis numérico con gráficos para obtener una mejor interpretación.
