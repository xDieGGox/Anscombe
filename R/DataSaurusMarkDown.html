Resumen del dataset

Aqu presentamos un resumen del dataset DataSaurus, en s lo que se hace es calcular las estadsticas para cada subgrupo del dataset. Con esto, podemos observar que las estadsticas son prcticamente iguales, aunque como vamos a ver posteriormente en esta prctica, en la visualizacin son completamente diferentes. De esta forma, demuestran lo que nos quiere decir el Cuarteto de Anscombe moderno. Para obtener estos resultados, se aplicaron las funciones group_by(dataset) para agrupar los datos segn cada conjunto y summarize() para calcular la media, desviacin estndar y correlacin.

datasaurus_dozen %>%
  group_by(dataset) %>%
  summarize(
    mean_x = mean(x),
    mean_y = mean(y),
    std_dev_x = sd(x),
    std_dev_y = sd(y),
    corr_x_y = cor(x,y)
  )

Grafica de CIRCLE

En esta parte, lo que se realiza es mostrar los puntos del dataset circle, para poder visualizar estos datos. Se puede observar que tienen forma de elipse o circunferencia, lo cual no es evidente solo con los valores estadsticos. Usamos la librera ggplot2 para generar el grfico, lo que nos ayuda a entender la distribucin de los datos, que en este caso forma un crculo. De esta manera, podemos complementar el anlisis numrico anterior con una visualizacin que revela la verdadera estructura de los datos, la cual no es evidente nicamente a travs de estadsticas.

ggplot(filter(datasaurus_dozen, dataset == "circle"), aes(x = x, y = y)) +
  geom_point(color = "#2C3E50") +
  theme_minimal() +
  labs(title = "Dataset: Circle")

Regresin lineal sobre CIRCLE

En esta parte aplicamos una regresin lineal en el dataset circle. Lo que se realiz fue filtrar los datos y luego, con la funcin lm(), pudimos obtener la recta que mejor se ajusta a los puntos. Luego, con ggplot2, mostramos los puntos y tambin, en rojo, la lnea que representa la regresin lineal. Esto nos sirvi para poder comparar el grfico que realizamos a mano, solo viendo la forma del crculo sin usar valores numricos, de tal forma que lo podemos comparar para ver si se parece o no al que hicimos de forma emprica.

# Filtramos el dataset
circle_data <- datasaurus_dozen %>%
  filter(dataset == "circle")

# Ajustamos el modelo de regresin lineal
modelo <- lm(y ~ x, data = circle_data)

ggplot(circle_data, aes(x = x, y = y)) +
  geom_point(color = "#2C3E50") +
  geom_smooth(method = "lm", se = FALSE, color = "red", linewidth = 1) +
  theme_minimal() +
  labs(title = "Regresin lineal - Dataset: Circle")
